
# Leave Management System - Frontend

This is the frontend client application for the Leave Management System. It is built using React, Vite, Tailwind CSS, and Axios. It interacts with the backend server running at http://localhost:5000.


## How to Run the Frontend

1. Navigate to the frontend directory:
   cd frontend
2. Install dependencies:
   npm install
3. Start the development server:
   npm run dev

Open http://localhost:5173 in your browser to view the application.
